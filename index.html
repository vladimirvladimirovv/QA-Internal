06 / Investigation Loop

1. Frontend

Check browser console/network tab for UI-specific crashes.

2. API

Isolate the request in Postman to confirm server-side failure.

3. Logs

Search for the Trace ID in Grafana to find the stack trace.

4. Database

Query DBeaver to ensure the data state matches the API claim.
